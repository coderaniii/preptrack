# PrepTrack

PrepTrack is a simple web platform designed to help students organize and track their coding preparation for placements.

## 🚀 Features

* Track DSA learning progress
* Structured learning paths
* Clean and simple UI
* Organized coding resources

## 🛠 Tech Stack

* HTML
* CSS
